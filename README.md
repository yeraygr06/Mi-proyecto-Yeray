# Dispensador automatico de lacasitos con pantalla informativa
Mi proyecto final es la creación de una maquina dispensadora de lacasitos mediante arduino.

# Abstract en Español
El proyecto consite en el diseño y construcción de un dispensador automático de lacasitos controlado mediante Arduino, capaz de dispensar y contar caramelos de forma autónoma. El sistema utiliza un motos paso a paso, un sensor fotointerruptor y una pantalla LCD para mostrar información al usuario. Se trata de un prototipo educativo, de bajo coste, que integra conocimientos de electrónica, programación y diseño mecánico.

# Abstract en Inglés
The project consists of the design and construction of an Arduino-controlled automatic candy dispenser capable of dispensing and counting sweets autonomously. The system uses a stepper motor, a photoelectric sensor and an LCD screen to display information to the user. It is a low-cost educational prototype that integrates knowledge of electronics, programming and mechanical design.
